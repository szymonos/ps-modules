# :material-powershell: psm-windows

Basic functions compatible with Windows PowerShell 5.1.

!!! note "Module Info"
    **Version:** 0.5.0 (beta) &middot; **PowerShell:** 5.1+ (Core and Desktop) &middot; **Dependencies:** none

## :material-function: Functions

| Function                        | Alias            | Description     |
| ------------------------------- | ---------------- | --------------- |
| `Get-LogMessage`                |                  | Format log msg  |
| `Invoke-CommandRetry`           |                  | Retry on errors |
| `Join-Str`                      |                  | Join to string  |
| `Test-IsAdmin`                  |                  | Check elevation |
| `Update-SessionEnvironmentPath` | `refreshenvpath` | Refresh PATH    |
