    Completers¤

    Tab-completion functions from Functions/completers.ps1 that query the Kubernetes API for live cluster data, enabling interactive argument completion.

    ArgK8sGetClusters¤

    Returns cluster names from kubectl config view. Completes the -Cluster parameter.

    Used by:

    Function / Alias Description Source
    Set-KubectlContext (kc) Switch context helper

    Used by:

    Function / Alias Description Source
    Set-KubectlContext (kc) Switch context helper
